# 台灣空氣品質互動地圖

一個現代化的互動式網頁應用，實時展示台灣各地的空氣品質監測資訊。

## 🌟 主要功能

### 📍 互動式地圖
- 在台灣地圖上標記所有空氣品質監測站
- 根據 AQI 等級用不同顏色標示（綠色、黃色、橙色、紅色、紫色）
- 點擊標記查看詳細信息
- 支持地圖縮放和平移

### 🔍 搜尋與過濾
- **快速搜尋**：按測站名稱或縣市搜尋
- **縣市篩選**：選擇特定縣市查看測站
- **鄉鎮篩選**：進一步按鄉鎮市區篩選
- **等級篩選**：按空氣品質等級篩選（良好、普通、不健康等）
- **一鍵重置**：快速清除所有篩選條件

### 📊 詳細資訊展示
每個測站顯示以下數據：
- **AQI 值**：空氣品質指數（0-500+）
- **空氣品質等級**：良好、普通、對敏感族群不健康、不健康、非常不健康、危害
- **主要污染物**：導致空氣品質下降的主要污染物
- **PM2.5**：細懸浮微粒濃度（μg/m³）
- **PM10**：懸浮微粒濃度（μg/m³）
- **O3**：臭氧濃度（ppb）
- **NO2**：二氧化氮濃度（ppb）
- **SO2**：二氧化硫濃度（ppb）
- **CO**：一氧化碳濃度（ppm）
- **風速**：當地風速（m/s）
- **更新時間**：數據最後更新時間

## 🎨 AQI 顏色編碼

| 顏色 | AQI 範圍 | 等級 | 建議 |
|------|---------|------|------|
| 🟢 綠色 | 0-50 | 良好 | 適合戶外活動 |
| 🟡 黃色 | 51-100 | 普通 | 敏感族群應減少戶外活動 |
| 🟠 橙色 | 101-150 | 對敏感族群不健康 | 敏感族群應避免戶外活動 |
| 🔴 紅色 | 151-200 | 不健康 | 所有人應減少戶外活動 |
| 🟣 紫色 | 201-300 | 非常不健康 | 所有人應避免戶外活動 |
| 🔴 深紅 | 301+ | 危害 | 所有人應留在室內 |

## 📦 技術棧

- **前端框架**：React 18 + TypeScript
- **構建工具**：Vite
- **地圖庫**：Leaflet
- **樣式**：CSS3 + 響應式設計
- **資料格式**：CSV

## 🔄 資料更新

空氣品質資料來自 `public/air_quality_data.csv`。若要更新資料：

1. 替換 `public/air_quality_data.csv` 檔案
2. 重新構建項目：`npm run build`
3. 重新部署
